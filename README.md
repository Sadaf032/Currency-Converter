# 💱 Currency Converter

A modern and responsive **Currency Converter Web App** that converts currencies using live exchange rates.  
Built with **HTML, CSS, and JavaScript** with API integration for real-time currency data.

---

## 🚀 Features

✨ Live currency conversion  
✨ Real-time exchange rates  
✨ Clean and responsive UI  
✨ Easy currency selection  
✨ Fast API-based results  
✨ User-friendly experience  

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure |
| CSS3 | Styling & Responsive Design |
| JavaScript | Logic & API Handling |
| Fetch API | Fetching exchange rates |
